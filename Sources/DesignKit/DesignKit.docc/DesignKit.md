# ``DesignKit``

A scalable, token-based design system and component library for SwiftUI.

## Overview

DesignKit provides a comprehensive set of design tokens, components, and utilities to build beautiful and consistent SwiftUI applications. Think of it as Bootstrap + Tailwind for iOS.

### Key Features

- **Token-Based Design**: Centralized design tokens for colors, spacing, typography, and more
- **Responsive Layout**: Bootstrap-inspired grid system with breakpoint support
- **Accessible**: Built-in accessibility support with Dynamic Type, VoiceOver, and minimum tap targets
- **Dark Mode**: Automatic light/dark theme support
- **Themeable**: Protocol-based theme system for easy customization
- **Multi-Platform**: iOS 16+, macOS 13+, tvOS 16+, watchOS 9+
